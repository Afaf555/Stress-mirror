# Огледало 🪞 — проверка на стрес за млади

Дигитално решение за предметот **ПНУВ**, инспирирано од студијата **HBSC** (Health Behaviour in School-aged Children, 2021/22).

Кратка, анонимна проверка преку која млади на возраст 11–15 год. забележуваат како се чувствуваат, гледаат како стојат во споредба со своите врсници од HBSC, и добиваат поддржувачки совети за справување.

## Што решава

Психосоматските симптоми на стрес кај адолесцентите растат со возраста, особено кај девојчињата (според HBSC, на 15 год. двојно повеќе девојчиња од момчиња пријавуваат повеќе чести симптоми). „Огледало" ја подига свесноста и го нормализира барањето поддршка.

## Технологии

- React 18 + Vite
- Без надворешни зависности, без backend
- Податоците се чуваат **локално на уредот** (localStorage) — следење на личен напредок низ време, ништо не се испраќа
- Реален **HBSC симптом-чеклист (HBSC-SCL, 8 ставки)** и реална скала на честота

## Функции

- Кратка проверка преку вистинскиот HBSC симптом-чеклист (8 ставки)
- Споредба со HBSC просеците по возраст и пол
- **Следење на напредок:** секоја проверка се чува; резултатот се споредува со претходниот пат („подобар си од вчера" / нежна порака ако е потешко)
- **Дневник:** запиши како се чувствуваш; записите се чуваат локално
- Преглед на трендот низ време + опција за бришење на сите податоци

## Како да се пушти локално

```bash
npm install
npm run dev
```

Отвори ја адресата што ќе се прикаже (обично http://localhost:5173).

## Build за продукција

```bash
npm run build
npm run preview
```

## Деплој (жив линк за предавање)

**Vercel / Netlify** — поврзи го GitHub repo-то, framework: Vite. Готово.

**GitHub Pages** — во `vite.config.js` смени `base` во `"/<име-на-repo>/"`, потоа:

```bash
npm run build
# качи ја содржината на dist/ преку gh-pages или GitHub Actions
```

## Извор на податоците

Референтните вредности за споредбата се изведени од WHO/HBSC International Report 2021/22 — *„A focus on adolescent mental health and well-being in Europe, Central Asia and Canada"*. Вредностите по ќелија (возраст × пол) се приближни, базирани на објавените трендови. За поточни бројки види ја HBSC Data Browser платформата.

## Важно

Ова е алатка за **самосвест**, не медицинска дијагноза. Апликацијата секогаш упатува дека разговор со возрасен на кого му се верува, училишен психолог или доктор може да помогне.
